# 目的
- RustのQUIC実装[quinn](https://github.com/quinn-rs/quinn)を使ってみること
- サービス立ち上げるひな型を作ること

# 使用法
- 環境変数を取得し、設定に反映する

| 設定名 | 初期値 | 内容 |
| ------| ------| ------|
| THE_CATCHER_IN_THE_LIE_BIND_PORT | 7777 | ポート番号 |
| THE_CATCHER_IN_THE_LIE_CERT_PATH | オレオレ | 証明書パス |
| THE_CATCHER_IN_THE_LIE_KEY_PATH | オレオレ | 秘密鍵パス |
